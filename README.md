# SENTINEL — Integrated Border Surveillance & Command Dashboard  

SENTINEL is a high-fidelity, browser-based **border monitoring and defence command system** designed for real-time surveillance, threat assessment, and tactical response operations.

The platform provides a centralized interface where multiple operational layers - intelligence, deployment, communication, and analytics — converge to support rapid decision-making in high-risk environments.

---

## Application Structure  

| File | Purpose |
|------|--------|
| `login.html` | Secure access interface featuring system boot and role-based entry |
| `dashboard.html` | Main operational dashboard integrating map, threats, deployments, AI insights, and analytics |

---

## Core Capabilities  

### Operational Command Layer  
- Consolidated real-time overview of system status  
- Key metrics including:
  - Active personnel strength  
  - Ongoing threat count  
  - Units deployed across sectors  
  - Mission success indicators  
- Dynamic UI feedback with alert escalation visuals  

---

### Geospatial Intelligence (Interactive Map)  
- Powered by **Leaflet.js** for real-time mapping  
- Sector-based visualization of national borders  
- Color-coded threat classification across regions  
- Interactive overlays for:
  - Unit positions  
  - Sector focus  
  - Threat hotspots  
- Multiple viewing modes (terrain / satellite style)  

---

### Threat Detection & Management  
- Continuous monitoring of intrusion events  
- Threats categorized by severity levels:
  - High (critical response required)  
  - Medium (monitor & evaluate)  
  - Low (routine observation)  
- Each threat contains:
  - Timestamp  
  - Location data  
  - Confidence metrics  
- Integrated action controls:
  - Immediate deployment  
  - Passive monitoring  
  - Threat dismissal  

---

### Tactical Deployment System  
- Intelligent allocation of:
  - Ground forces  
  - Surveillance drones  
  - Specialized response units  
- Live deployment tracking with:
  - Movement progression  
  - Status transitions (in transit / active / standby)  
- Visual feedback on mission execution  

---

### AI Decision Support Module  
- AI-driven intelligence engine  
- Provides:
  - Predictive threat analysis  
  - Strategic deployment suggestions  
  - Risk scoring based on current system state  
- Enhances situational awareness for command-level decisions  

---

### Secure Communication Layer  
- Real-time command messaging interface  
- Structured broadcast system for:
  - Emergency directives  
  - Standby instructions  
  - Silent operations  
- Multi-channel communication simulation across units  

---

### Analytics & Performance Monitoring  
- Built using **Chart.js** for dynamic visualization  
- Tracks:
  - Threat frequency trends  
  - Deployment efficiency  
  - Mission success ratios over time  
- Enables data-driven evaluation of operations  

---

### Environmental Intelligence  
- Environmental parameters affecting missions:
  - Weather conditions  
  - Terrain difficulty  
- Displays operational impact indicators for each factor  

---

### Cybersecurity Monitoring  
- System integrity visualization  
- Intrusion attempts and logs  
- Network health indicators for secure operations  

---

### Activity & Event Logging  
- Centralized log of all system events:
  - Threat detections  
  - Deployments  
  - Communication broadcasts  
  - System updates  
- Chronological tracking for audit and review  

---

## Getting Started  

1. Download or clone the repository  
2. Open `login.html` in any modern browser  
3. Select a role and proceed to access the dashboard  

---

## Technology Stack  

- **Frontend:** HTML5, CSS3, Vanilla JavaScript  
- **Mapping:** Leaflet.js  
- **Data Visualization:** Chart.js  
- **UI Design:** Custom military-inspired interface using:
  - Orbitron  
  - Rajdhani  
  - Share Tech Mono
